Copy Cell/Column Value Module

Description

This module allows users to easily copy the value of a cell or an entire column in Odoo's list view with just one click. This feature is extremely useful when you need to quickly grab data from a specific cell or copy the entire data of a column for use elsewhere, such as pasting into Excel or other applications.

Save time: No need to manually select individual values or export unnecessary data.

Installation and Activation Guide

To use the module, you need to install and activate it in Odoo:

  1. Go to the Apps menu.

    2. Step 1: Locate the Apps menu

  2. Search for the module "Copy Cell/Column Value" or its technical name (if available).
  3. Click the Install or Activate button.

    4. Step 2 & 3: Search and activate the module

Features and Results

Improved User Experience: Quick and intuitive copy operations.

1. Copy Icon Appears on Hover

When you hover over a cell in the list view, a small clipboard icon appears, indicating that you can copy the cell's value.

Copy icon appears when hovering over a cell

2. Copy Cell Value

Click the clipboard icon to copy the value of that cell to the clipboard.

Click to copy the cell value

3. Copy Entire Column

Click the column header to copy all values in that column (for the records currently displayed on the page).

Click the header to copy the entire column
